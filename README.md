## Installation

### Insatll nodejs 7.x version
```bash
$ wget -qO- https://deb.nodesource.com/setup_7.x | sudo bash -
$ sudo apt-get install -y nodejs
```
### Clone the code
### Move to project folder
### Install npm dependencies
```bash
$ npm i
```
### Build project
```bash
$ npm run production
```
### Run project
```
$ forever start server.js
```
